# I Wanna 游戏项目

基于Web技术开发的经典I Wanna风格平台跳跃游戏，包含完整的关卡编辑器。

## 🎮 项目特色

- **经典I Wanna玩法** - 高难度平台跳跃挑战
- **关卡编辑器** - 可视化关卡设计工具
- **静态资源** - 精美的像素风格素材
- **Web原生** - 基于HTML5 Canvas技术

## 🛠️ 技术栈

- **前端**: HTML5 + Canvas + JavaScript
- **图形**: 2D像素风格渲染
- **物理**: 自定义重力和碰撞系统
- **资源**: PNG静态图片素材

## 📁 项目结构

```
YouWanna/
├── src/                    # 静态资源文件
│   ├── *.png              # 游戏素材图片
│   ├── main.lua           # Lua游戏逻辑(备用)
│   ├── GameScene.lua      # 游戏场景(备用)
│   ├── AIArt.lua          # 资源加载器(备用)
│   └── ObstacleSystem.lua # 障碍物系统(备用)
├── iwanna_replica.html    # 主游戏文件
├── level_editor.html      # 关卡编辑器
├── project.json           # 项目配置
└── README.md             # 项目说明
```

## 🎮 操作说明

### 游戏操作
- **WASD** 或 **方向键** - 移动
- **空格键** - 跳跃

### 编辑器操作
- **左键** - 绘制选中元素
- **右键** - 擦除元素
- **工具栏** - 选择不同的绘制工具

## 🚀 快速开始

1. 启动本地服务器: `python -m http.server 8080`
2. 打开游戏: `http://localhost:8080/iwanna_replica.html`
3. 打开编辑器: `http://localhost:8080/level_editor.html`
